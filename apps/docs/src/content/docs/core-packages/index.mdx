Core Packages are the official packages developed and maintained for the Recall
project. They provide shared building blocks used throughout Recall—from local
development and testing to production.

These packages keep common functionality consistent, reusable, and reliable
across the application and the wider Recall ecosystem.

## Packages

- [Marketplace Interface](/core-packages/marketplace-interface/) — the client
  interface for communicating with Recall-compatible marketplace servers.
